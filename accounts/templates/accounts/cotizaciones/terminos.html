Guía Básica para Usar Etiquetas HTML en tus Términos y Avisos


¡Hola! Si necesitas agregar formato a tu texto para los términos y condiciones, puedes usar algunas etiquetas HTML básicas. Aquí te explicamos cómo hacerlo de manera sencilla.
  1. Negritas
    2. Para resaltar texto en negritas, usa la etiqueta '<strong>'. Cualquier texto entre '<strong>' y '</strong>' se verá en negritas.
    Ejemplo: <strong> Este texto se verá en negritas </strong>
  2. Cursiva
    3.  Para poner texto en cursiva, usa la etiqueta '<em>'. El texto entre '<em>' y '</em>' se verá en cursiva.
    Ejemplo: <em> Este texto se verá en cursiva </em>
  3. Subrayado
    4.  Para subrayar texto, usa la etiqueta '<u>'. El texto entre '<u>' y '<u>' estará subrayado.
    Ejemplo: <u> Este texto estará subrayado </u>
  4. Saltos de línea
    5.  Para comenzar un nuevo párrafo, usa la etiqueta '<br>'. Esto insertará un salto de línea.
    Ejemplo:
    Este es el primer párrafo.<br>
    Este es el segundo párrafo después del salto de línea.
  5. Viñeta
    6.  Para crear listas con viñetas, usa las etiquetas '<ul>' y '<li>'. La etiqueta '<ul>' inicia una lista no ordenada (con viñetas) y cada elemento de la lista va entre '<li>' y '</li>'.
    Ejemplo:
    <ul>
    • <li> Primer ítem de la lista </li>
    • <li> Segundo ítem de la lista </li>
    • <li> Tercer ítem de la lista </li>
    </ul>
  6. Comentarios dentro del texto
    7. Para comentar el texto puesto que alomejor necesitan aclarar el porque de alguna seccion del texto puede usarse lo siguiente '<!-- Comentario aquí -->', esto solo es para el campo para ingresar texto, no aparecera en el documento PDF generado.
